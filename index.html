La formation nous fait découvrir deux aspects essentiels des langages informatiques :

Le "front-end" sert à créer la structure (HTML), la mise en forme (CSS) et l'animation (JavaScript) du site.

Le "back-end" permet de rendre le site dynamique via l'utilisation de fonctionnalités à partir du langage informatique PHP.

De plus, l'utilisation d'outils ("framework") permet d'optimiser le travail du développeur web. (Symfony pour PHP, Jquery pour JavaScript, Bootstrap pour le responsive etc)

Il existe également des outils collaboratifs tels que Framapad ("bloc note numérique") qui permet aux développeurs de communiquer entre eux sur un projet. Le trello est un agenda qui permet d'aménager un planning de travail.

Github a pour but de comiter ("enregistrer") son travail entre collaborateurs.
